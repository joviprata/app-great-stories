## Installation
To install and run this application, clone this repository and, in its root folder, run the following command in the terminal:

```bash
npm start
```
Then open your browser at http://localhost:4200/

To visualize the mobile layout, enable your browserâ€™s responsive mode (e.g., `Ctrl + Shift + M` on Chrome / Firefox, or `Cmd + Option + R` on Safari).

> **Note:** The initial loading screen is a static placeholder and does not perform any action. To proceed, click on the Pooh Bear image to navigate to the next screen.
